# Java Learning Repository

This repository is a structured journey of learning Java from fundamentals to advanced concepts.  
It contains practice code, small projects, and experiments built while learning the language step by step.

The goal is not just to write code, but to understand how Java works internally and how to design clean, maintainable programs.

---

## 🎯 Purpose

- Learn Java from zero to advanced level
- Practice Object-Oriented Programming (OOP)
- Build small real-world console applications
- Improve problem-solving and coding discipline
- Prepare for backend and software engineering fundamentals

---

## Topics Covered

### 1. Java Basics
- Variables and data types
- Operators
- Conditional statements (if / switch)
- Loops (for / while)

### 2. Methods & Functions
- Method creation and usage
- Parameters and return types
- Method overloading

### 3. Object-Oriented Programming (OOP)
- Classes and objects
- Encapsulation
- Constructors
- Inheritance
- Polymorphism (basic level)

### 4. Data Structures (Java Collections)
- ArrayList
- Arrays
- Basic data handling

### 5. Console Applications
- Menu-driven programs
- User input handling (Scanner)
- Simple system design (booking, management systems, etc.)

---

## Projects In

This repository may include small experimental projects such as:

- Room reservation system (console-based)
- Student management system
- Simple calculator
- Practice exercises and coding challenges

Each project is meant to reinforce core Java concepts.
